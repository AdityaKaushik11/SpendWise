## ✨ Overview

**SpendWise** is a full-stack expense tracking application built to make personal finance management more intelligent and practical.

Instead of being just another basic tracker, SpendWise combines:

- **expense management**
- **budget monitoring**
- **interactive visual analytics**
- **AI-generated financial recommendations**

The goal is simple: help users understand **where their money goes**, identify bad spending patterns, and make smarter financial decisions.

## 🧠 AI Insights Feature

One of the strongest parts of SpendWise is its **AI Spending Insights** module.

It analyzes the user's recent expenses and generates structured insights such as:

- **Top spending category**
- **Spending pattern summary**
- **Actionable recommendations**
- **Budget improvement suggestions**

This makes SpendWise more than a tracker — it behaves like a lightweight **personal finance assistant**.

## 🧩 Core Features

### 1. Expense Management
- Add expenses with amount, category, date, and note
- Mark expenses as **recurring** or **one-time**
- Edit and delete transactions
- Search expenses by note or category

### 2. Budget Goals
- Set monthly spending limits for categories
- Track category performance against goals
- Build better spending discipline

### 3. Interactive Analytics
- Category breakdown using pie charts
- Daily monthly totals using bar charts
- Multi-month spending trends using line charts

### 4. AI-Powered Recommendations
- Detect heavy-spending categories
- Summarize financial behavior
- Suggest smarter decisions based on transaction history

### 5. Export Functionality
- Export expense data in **CSV format**

### 6. Authentication & Security
- User signup and login
- Secure cloud-based database
- Personal data isolation using Supabase policies
